Isibia Dashboard Messages - leave messages in the website control panel for all users.

== Description ==
Isibia Dashboard Messages - leave messages in the website control panel for all users. You can leave instructions,
reminders and wish Happy New Year.

= Features =
* Creating messages for Admin Panel users
* The message can be closed or it cannot be closed.
* You can specify the date period in which the message should be displayed.
